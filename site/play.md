# Play Demo

<a href="{{ site.baseurl }}/play/" class="btn">Launch Game</a>

The demo runs directly in your browser — no download needed. Use arrow keys or WASD to move, Enter to interact, and Escape for the menu.

## Controls

| Action | Keys |
|--------|------|
| Move | Arrow keys / WASD |
| Interact | Enter / Space |
| Menu | Escape |
| Debug teleport | 1 (Havenwood), 2 (Ruins) |
| Test battle | B |
