# GPT Prompt: Character Pose Sprite

## Template

A 2D game sprite of [CHARACTER_DESCRIPTION] in a [POSE_DESCRIPTION].

**Proportions:** Chibi, 3-head tall, oversized head relative to body.

**Style:**
- Soft cel-shading with clean dark outlines
- High-quality detailed rendering like a premium mobile JRPG overworld sprite
- Solid white background for easy extraction
- No shadows, scenery, or text

## Variables to Replace
- [CHARACTER_DESCRIPTION]: Name, hair, eyes, clothing — e.g., "a male character with spiky red hair, amber eyes, wearing navy medieval tunics"
- [POSE_DESCRIPTION]: What the character is doing — e.g., "lying unconscious face-down with limbs slightly splayed"

## After Generation
1. Save to: `games/chimera/drafts/sprites/characters/[name]_[pose].png`
2. Move to: `web/public/sprites/characters/[name]_[pose].png`
